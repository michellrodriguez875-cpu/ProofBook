# 📚 Library Manager on Solana

Built by **Ctrl+Z Squad** 🚀

## 🧠 Description

This project is a simple CRUD program built on the Solana blockchain using the Anchor framework.
It allows users to register, update, and delete books in a decentralized way.

## ⚙️ Features

* Add a new book
* Update book availability
* Delete a book
* Store data on-chain

## 🛠 Tech Stack

* Solana
* Rust
* Anchor

## 🚀 How to Run

### Build

anchor build

### Test

anchor test

### Deploy

anchor deploy

## 📦 Program Structure

* **BookAccount** → stores book data
* **Instructions** → add, update, delete

## 🌐 Network

Deployed on Solana Devnet
